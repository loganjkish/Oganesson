# Oganesson
A project to try and make the heaviest HTML file possible, for funsies.

## Running
Simply go to the website oganesson.loganjkish.com (NOT UP YET) to download a copy!
Opening the file can be very slow, please be patient. Once loaded, as indicated by the particle background, select a game icon to continue, which will have an even longer load time.

## Building
Simply clone the repo, install dependencies, and run build.py to get your very own copy!

## Contributing
Not every game can be put into Oganesson easily, but the best way is to 
- Clone the game
- Patch it if necessary
- Use register_game.py to add it in to the build
- Remove the temp clone directory

Feel free to look at the the nzp.py script to get a start!
Also feel free to make a pull request and add more games to the public Oganesson!
